Memory hints

90° ↔ π/2 changes sin ↔ cos
180° ↔ π changes the quadrant sign
360° ↔ 2π returns to the same direction

These are reduction identities. First decide whether the function changes to its co-function, then decide the sign using the quadrant.

  • 90° ± θ and 270° ± θ usually change sin ↔ cos, tan ↔ cot.
  • 180° ± θ and 360° ± θ usually keep the same function type.
  • Use ASTC/quadrant signs to decide plus or minus.
