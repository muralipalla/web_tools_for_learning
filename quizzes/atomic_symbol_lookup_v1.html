How to use

This lookup tool is designed for quick JEE-style chemistry revision. Enter a symbol, name, or atomic number and press Search. The result appears as a periodic-table-style tile.

Symbol → Atomic number + atomic weight
Example: Fe → Z = 26, atomic weight = 55.845 u
  • Atomic number is the number of protons.
  • Atomic weight is the average atomic mass in atomic mass units.
  • Bracketed values such as [222] usually indicate no stable standard atomic weight.
